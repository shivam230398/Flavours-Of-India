## Introduction
`Flavours of India` is a restaurant website that showcases a variety of Indian cuisines. It provides an interactive menu, reservation system, and information about the restaurant.

## Features
- Interactive menu with detailed descriptions of each dish.
- User-friendly reservation system.
- Contact form for inquiries.
- Responsive design for mobile and desktop views.

## Installation
1. Clone the repository:  
   `git clone https://github.com/shivam230398/flavours-of-india.git`
2. Navigate to the project directory:  
   `cd flavours-of-india`
3. Install dependencies:  
   `npm install`
4. Start the development server:  
   `npm start`

## Usage
- Visit the website at `http://localhost:3000` to view the restaurant's offerings.
- Use the reservation form to book a table.
